# Installation
## Dependencies

```
pip install stable_baselines3
pip install torch
pip install gym
```
If you install above packages, please install `Bullet-Safety-Gym`

```
git clone https://github.com/SvenGronauer/Bullet-Safety-Gym.git

cd Bullet-Safety-Gym

pip install -e .
```

# Getting Started
```
cd Ball-Run
python impulse_separated.py
```
